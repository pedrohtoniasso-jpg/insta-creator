# Roadmap

## Phase 1 — System specification
- Define the shared skill suite boundaries.
- Define the project-spec contract.
- Define the post storage schema and folder naming convention.
- Define the bootstrap responsibilities.

## Phase 2 — Bootstrap package
- Build the reusable bootstrap for the skill suite.
- Add setup/validation steps for new environments.
- Add templates and defaults for new projects.

## Phase 3 — Content workflow skills
- Implement the brief generation skill.
- Implement the cards/JSON skill.
- Implement the caption/CTA/hashtags skill.
- Implement the approval-package skill.

## Phase 4 — Project integration
- Add project-specific spec loading.
- Connect cron-fed ideas to the workflow.
- Connect Trello traceability and publication hooks.

## Phase 5 — Operational hardening
- Verify folder outputs.
- Verify deterministic naming.
- Verify bootstrap reproducibility in a clean environment.
