# last30days

Use this skill when the task is to discover what is happening now in a niche and turn it into post ideas.

## Typical sources
- X/Twitter
- Reddit
- YouTube
- TikTok
- Web search

## Goal
Find subjects, narratives, formats, and questions that appear repeatedly in the last few weeks and summarize them into actionable ideas.

## Expected output
- 3 to 5 trending themes
- why each theme is gaining traction
- which theme makes the most sense for the brand/segment
- which format works best
- one main recommendation to approve

## Rules
- Prioritize recent signals, not old content.
- Do not copy; extract patterns.
- If signals are weak, say the search was inconclusive.
- For brand posts, connect the topic to the sector context before suggesting publication.
